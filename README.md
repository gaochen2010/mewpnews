# 全球高空作业平台租金价格跟踪系统

一个用于跟踪全球主要地区高空作业平台月度租金价格水平的Web应用。

## 功能特性

### 📊 核心功能
- **设备类型监控**: 支持剪叉式(6m)、曲臂式(14m)、直臂式(18m)、桅柱式(4m)四种设备类型
- **多地区支持**: 全球平均、亚洲、欧洲、北美及主要国家（中国、美国、英国、德国、日本）
- **价格走势图表**: 直观展示12个月的价格变化趋势
- **数据表格**: 详细的数据表格，包含环比和同比变化
- **设备对比**: 多设备类型价格对比分析
- **数据导出**: 支持导出CSV格式数据

### 📈 数据展示
- 当前月租金及环比变化
- 平均月租金（过去12个月）
- 最高/最低月租金及对应月份
- 交互式价格走势图
- 详细数据表格

## 使用方法

### 方法一：使用Python启动服务器（推荐）

1. 确保已安装Python 3
2. 在终端中运行：
```bash
python3 start_server.py
```
3. 浏览器会自动打开页面，或手动访问：`http://localhost:8000/高空作业平台租金跟踪.html`

### 方法二：直接打开HTML文件

1. 双击 `高空作业平台租金跟踪.html` 文件
2. 文件会在默认浏览器中打开

### 方法三：使用其他HTTP服务器

如果安装了Node.js，可以使用：
```bash
npx http-server -p 8000
```
