# 建筑业周报生成工具使用说明

## 📋 文件说明

1. **建筑业周报_2025-12-02.html** - 周报HTML模板（包含完整格式，待填入新闻内容）
2. **新闻数据模板.json** - 结构化的新闻数据模板，便于填写
3. **新闻搜集指南.md** - 详细的新闻来源和搜集方法指南
4. **generate_report.py** - 从JSON数据自动生成HTML周报的脚本
5. **update_weekly_report.py** - 手动更新HTML周报的辅助脚本

## 🚀 快速开始

### 方法一：直接编辑HTML文件（推荐用于少量更新）

1. 打开 `建筑业周报_2025-12-02.html`
2. 找到对应的section（如 `<section id="sec1">`）
3. 在 `<tbody>` 标签内添加新闻条目，格式如下：

```html
<tr>
  <td>12月1日</td>
  <td><strong>美国 — 某大型数据中心项目</strong></td>
  <td>项目详情描述...</td>
  <td>影响分析...</td>
</tr>
```

### 方法二：使用JSON模板 + 自动生成（推荐用于批量更新）

1. **填写JSON模板**
   - 打开 `新闻数据模板.json`
   - 按照格式填写各部分的新闻数据
   - 保存文件

2. **生成HTML周报**
   ```bash
   python3 generate_report.py 新闻数据模板.json 建筑业周报_2025-12-02_完整版.html
   ```

## 📰 新闻搜集建议

由于无法直接访问互联网，建议通过以下方式搜集新闻：

### 1. 使用新闻聚合工具
- Google News（搜索 "construction industry" + 日期范围）
- Feedly（订阅建筑行业RSS源）
- 行业专业网站（见 `新闻搜集指南.md`）

### 2. 主要新闻来源
- **国际媒体**：Construction Week, ENR, Building Design+Construction
- **企业官网**：大型建筑企业的新闻发布页面
- **政府网站**：基础设施项目公告
- **行业组织**：建筑行业协会的新闻发布

### 3. 搜索关键词
- 英文：construction project, infrastructure investment, MEWP, aerial work platform
- 中文：建筑项目、基础设施、高空作业平台、工程机械

## 📝 数据填写格式

### JSON格式示例

```json
{
  "section1_projects": [
    {
      "date": "12月1日",
      "region": "<strong>美国 — 某数据中心项目</strong>",
      "details": "投资额XX亿美元，预计2027年完工...",
      "impact": "将带动电动剪叉和高空作业平台需求..."
    }
  ]
}
```

### HTML格式示例

```html
<tr>
  <td>12月1日</td>
  <td><strong>美国 — 某数据中心项目</strong></td>
  <td>投资额XX亿美元，预计2027年完工...</td>
  <td>将带动电动剪叉和高空作业平台需求...</td>
</tr>
```

## ⚠️ 注意事项

1. **时效性**：确保新闻发生在报告周期内（11月25日-12月2日）
2. **准确性**：优先使用官方发布和权威媒体报道
3. **完整性**：每个新闻条目应包含日期、地区、关键信息和影响分析
4. **格式规范**：
   - 日期使用中文格式（如：12月1日）
   - 重要名称使用 `<strong>` 标签加粗
   - 保持表格列数一致

## 🔧 故障排除

### 问题：Python脚本运行出错
- 检查Python版本（需要Python 3.6+）
- 确保JSON文件格式正确
- 检查文件路径是否正确

### 问题：生成的HTML格式不对
- 检查JSON数据中的HTML标签是否正确转义
- 确保表格列数与模板一致
- 查看控制台错误信息
